# Covid-19 Tracker

An archived early frontend project built to practice React Hooks, responsive UI development, real-time API integration, data visualization, interactive maps, and Firebase Hosting deployment. The application provides a Covid-19 tracking dashboard that displays global and country-level statistics through charts, maps, and selectable country data.

## Live Demo

https://covid19-tracker-6ae25.web.app/

## Project Overview

This project was created as an early React practice project to strengthen frontend development and data visualization skills. It focuses on fetching real-time public health data, presenting statistics through charts, and using an interactive map to display country-level Covid-19 information.

## Tech Stack

- React
- React Hooks
- JavaScript
- Material UI
- Fetch API
- react-chartjs-2
- react-leaflet
- Firebase Hosting

## Features

- Mobile-responsive Covid-19 tracking dashboard
- Real-time Covid statistics retrieved from the disease.sh API
- Line chart visualization for statistical trends
- Interactive map for geographic data display
- Country dropdown menu for selecting and viewing country-specific statistics
- Firebase cloud deployment for public hosting

## What I Practiced

- Building React applications with Hooks
- Fetching and rendering real-time API data
- Creating data visualizations with react-chartjs-2
- Building map-based UI features with react-leaflet
- Using Material UI components for layout and styling
- Deploying a React application to Firebase Hosting
